

# Transpose of a Matrix

## Description
This function computes the **transpose** of a given matrix.  
The **transpose** of a matrix is obtained by flipping its rows and columns.

## Example

### **Input:**
```python
a = [[1, 2, 3], [4, 5, 6]]
```
### **Input:**
```
[[1, 4], [2, 5], [3, 6]]
```

### **Reasoning** 

The transpose of a matrix is obtained by flipping rows and columns.
